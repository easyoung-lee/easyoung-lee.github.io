### 소문자로 된 문자가 입력되면 그 단어의 가운데 문자를 출력하는 프로그램을 작성하시오(단 단어의 길이가 짝수일 경우 가운데 2개의 문자를 출력한다)

---

# example
설명
```python
#Greedy Algorithmn 상하좌우 문제
n = int(input())
plans = input().split()
x, y = 1, 1

dx = [0, 0, 1, -1]
dy = [1, -1, 0, 0]
move_types = ['R', 'L', 'S', 'N']

for plan in plans:
  for i in range(len(move_types)):
    if plan == move_types[i]:
      nx = x+dx[i]
      ny = y+dy[i]
  if nx<1 or ny<1 or nx>n or ny>n:
    continue
  x = nx
  y = ny

print(x, y)
```
